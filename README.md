# diceware-rust

My first attempt at coding in Rust.

Read more about [Diceware](http://world.std.com/%7Ereinhold/diceware.html).

## Build

Install Rust & Cargo, I use Rustup.

```bash
# Build
cargo build --release

# "Install" to your path
cp target/release/diceware-rust /where/ever/bin/
```

## Use

```bash
diceware-rust
```

## Help

```bash
diceware-rust -h
```
